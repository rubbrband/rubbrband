## Official Releases

Rubbrband is available on PyPi, so to install it just run the following

```
pip install rubbrband
```

This will install the Python dependencies required for Rubbrband. Rubbrband also depends on Docker to make training just 1 line of code. 


## Docker installation

Rubbrband requires Docker to be installed. Installation instructions found [here](https://docs.docker.com/get-docker/).
